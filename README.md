## 🎯 What is CareerPilot AI?

CareerPilot AI is a multi-agent system that helps students and early professionals go from:

**Current Situation → Career Goal → Action Plan → Resume → Projects → Interview Prep**

A first-year CS student can upload their CV, describe their career goal, and receive a complete, personalized career development report — automatically generated by 5 specialized AI agents working in sequence.

## 🚀 Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/ahmedyaloufi-gif/careerpilot-ai.git
cd careerpilot-ai
```

### 2. Create virtual environment
```bash
uv venv --python 3.11
source .venv/bin/activate
```

### 3. Install dependencies
```bash
uv pip install google-adk streamlit PyPDF2 python-dotenv mcp
```

### 4. Set up API key
Create a `.env` file in the root directory:
GOOGLE_API_KEY=your_gemini_api_key_here
Get your free API key at [aistudio.google.com](https://aistudio.google.com)

### 5. Run via CLI
```bash
python main.py
```

### 6. Run via Streamlit UI
```bash
streamlit run frontend/app.py
```

## 🔒 Security Design

CareerPilot AI takes user privacy seriously:

1. **Permission Layer** — Users explicitly approve access to their data before any processing begins
2. **PII Masking** — Emails, phone numbers, and addresses are detected and masked before being sent to AI agents
3. **Data Vault** — Original values stored locally, never sent to external services
4. **No Data Storage** — Reports are saved locally only, never to external servers

```python
# Example: PII is masked before sending to agents
masked_cv, vault = mask_personal_data(cv_text)
# ahmed@email.com → [EMAIL_1]
# +966 50 000 0000 → [PHONE_1]
```

## 🔌 MCP Integration

CareerPilot uses a Filesystem MCP server with 3 tools:

| Tool | Description |
|---|---|
| `read_cv_from_file` | Reads PDF/TXT CV from disk |
| `save_report_to_file` | Saves full report to outputs/ |
| `list_uploaded_files` | Lists available CV files |

```python
# Example MCP usage
result = read_cv_from_file("uploads/my_cv.pdf")
# Returns: content + metadata (file size, word count, pages)
```
